To run the package:
1) unzip "ar_week5_test.zip" in a workspace
2) open a terminal
3) change the path to workspace
   cd [workspace_name]
4) build a catkin workspace
   catkin_make
   source devel/setup.bash
5) run the launch file to start
   roslaunch ar_week5_test cubic_traj_gen.launch

If the plot doesn't show any topic, 
1) Input topic name on the top left
2) press the green plus bottom
The topic should be on the plot.

If the node or the topic doesn't show on the graph,
1) press the left top refresh bottom
The topic will show if it is published.
